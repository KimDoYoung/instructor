HTML Semantic 태그 예제

이 예제는 HTML의 다양한 semantic 태그들을 소개하는 예제입니다. 아래는 간단한 설명과 함께 다양한 태그들을 사용한 예시가 있습니다.

1. 목록 태그

<ul>: 순서 없는 목록을 정의
  • <ol>: 순서 있는 목록을 정의
  • <li>: 목록 항목을 정의

    • 2. 링크 태그

    웹 페이지 간의 이동을 위한 링크를 생성합니다.

    • <a>: 하이퍼링크를 정의

    이곳은 예제 웹사이트입니다.

    3. 이미지 태그

    웹 페이지에 이미지를 삽입합니다.

    • <img>: 이미지를 정의

    4. 테이블 태그

    표를 작성합니다.

    • <table>: 테이블을 정의
    • <tr>: 테이블 행을 정의
    • <th>: 테이블 헤더를 정의
    • <td>: 테이블 데이터를 정의

    5. 양식 태그

    사용자로부터 정보를 수집하기 위한 양식을 정의합니다.

    • <form>: 양식을 정의
    • <input>: 입력 필드를 정의
    • <textarea>: 여러 줄의 텍스트 입력 필드를 정의
    • <select>: 드롭다운 목록을 정의

    6. div 태그

    블록 레벨 요소로, 콘텐츠를 그룹화하거나 스타일을 적용하는 데 사용됩니다.

    div는 굉장히 많이 쓰이는 태그입니다
